* 概述

Fuzzy Search 是一个基于最长公共子序列 (LCS) 算法的 Emacs 模糊搜索工具。它可以在当前窗口或选中的区域内进行模糊匹配，并通过不同颜色高亮显示匹配结果。

* 功能特性

- 基于 LCS 算法的 Token 级模糊匹配
- 支持自定义阈值过滤匹配结果
- 三种状态高亮显示：
  - =equal=（匹配）：使用 =diff-index= 面
  - =less=（多余）：使用 =diff-removed= 面
  - =more=（缺少）：以斜体形式在后方显示
- 自动定位到最佳匹配位置并居中显示

* 安装

** 依赖

- Emacs 27.0+ (需要 =cl-lib= 和 =eieio=)

** 安装步骤

+ 克隆项目到你自己的 Emacs 配置目录下：
#+BEGIN_SRC sh
  git clone https://github.com/ginqi7/fuzzy-search ~/.emacs.d/lisp/fuzzy-search
#+END_SRC

+ 把克隆的路径添加到 =load-path= 当中
#+BEGIN_SRC emacs-lisp
  (add-to-list 'load-path "~/.emacs.d/lisp/fuzzy-search")
#+END_SRC

* 配置

#+BEGIN_SRC emacs-lisp
  (require 'fuzzy-search)
#+END_SRC

** 变量

| 变量名                     | 默认值 | 说明                                      |
|----------------------------+--------+-------------------------------------------|
| =fuzzy-search-recenter-line= | 10     | 匹配后重新居中时，目标行所在的窗口行号    |

可以通过 =setq= 或 =customize-variable= 修改配置：

#+BEGIN_SRC emacs-lisp
  (setq fuzzy-search-recenter-line 15)
#+END_SRC

* 命令

| 命令              | 说明                                                                 |
|-------------------+----------------------------------------------------------------------|
| =fuzzy-search=      | 在当前窗口或选中区域内执行模糊搜索，支持交互式阈值参数 (0-1，默认 0.5) |

绑定快捷键示例：

#+BEGIN_SRC emacs-lisp
  (global-set-key (kbd "C-c s") 'fuzzy-search)
#+END_SRC

* 使用方法

** 基本用法

1. 激活一个区域（可选），或者直接在整个窗口范围内搜索
2. 运行 =M-x fuzzy-search=
3. 输入搜索查询字符串
4. （可选）输入阈值（0-1 之间，默认 0.5）

** 搜索示例

假设当前缓冲区内容为：
#+BEGIN_EXAMPLE
  (defun fuzzy-search-query-tokens (str)
    "Generate token list from string.")
#+END_EXAMPLE

搜索 =fz query= 会：
- 匹配包含 =fz= 和 =query= 相关 token 的区域
- 高亮显示匹配部分
- 将光标移动到匹配结束位置

* 工作原理

1. 将搜索查询和待搜索区域分别切分为 token 列表（字母数字组合）
2. 使用 LCS 算法计算两个 token 列表的差异
3. 滑动窗口找到最佳匹配位置
4. 根据阈值判断是否显示结果
