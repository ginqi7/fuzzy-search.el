* Overview

Fuzzy Search is an Emacs fuzzy search tool based on the Longest Common Subsequence (LCS) algorithm. It performs fuzzy matching within the current window or selected region, and highlights match results with different colors.

* Features

- Token-level fuzzy matching based on LCS algorithm
- Customizable threshold for filtering match results
- Three-state highlighting:
  - =equal= (matched): uses =diff-index= face
  - =less= (extra): uses =diff-removed= face
  - =more= (missing): displayed in italic after the text
- Automatically positions and centers the best match

* Prerequisites

** Requirements

- Emacs 27.0+ (requires =cl-lib= and =eieio=)

* Installation

+ Clone the project to your Emacs configuration directory:
  #+BEGIN_SRC sh
    git clone https://github.com/ginqi7/fuzzy-search ~/.emacs.d/lisp/fuzzy-search
  #+END_SRC

+ Add the path to =load-path=:
  #+BEGIN_SRC emacs-lisp
    (add-to-list 'load-path "~/.emacs.d/lisp/fuzzy-search")
  #+END_SRC

* Configuration

#+BEGIN_SRC emacs-lisp
  (require 'fuzzy-search)
#+END_SRC

** Variables

| Variable                   | Default | Description                                        |
|----------------------------+---------+----------------------------------------------------|
| =fuzzy-search-recenter-line= | 10      | Line number for recentering after a successful match |

Configure with =setq= or =customize-variable=:

#+BEGIN_SRC emacs-lisp
  (setq fuzzy-search-recenter-line 15)
#+END_SRC

** Commands

| Command        | Description                                                    |
|----------------+----------------------------------------------------------------|
| =fuzzy-search=   | Perform fuzzy search in current window or region with optional threshold (0-1, default 0.5) |

Example key binding:

#+BEGIN_SRC emacs-lisp
  (global-set-key (kbd "C-c s") 'fuzzy-search)
#+END_SRC

* Usage

** Basic Usage

1. Activate a region (optional), or search within the entire window
2. Run =M-x fuzzy-search=
3. Enter the search query string
4. (Optional) Enter a threshold value between 0 and 1 (default 0.5)

** Search Example

Given the following buffer content:
#+BEGIN_EXAMPLE
  (defun fuzzy-search-query-tokens (str)
    "Generate token list from string.")
#+END_EXAMPLE

Searching for =fz query= will:
- Match the region containing tokens related to =fz= and =query=
- Highlight the matched parts
- Move cursor to the end of the match

* How It Works

1. Tokenize the search query and the target region into token lists (alphanumeric sequences)
2. Use LCS algorithm to compute differences between the two token lists
3. Slide a window to find the best match position
4. Display results if they meet the threshold requirement
